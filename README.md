# Museum node-express

1.  Install the dependencies

    ```bash
    yarn
    ```

        or

    ```bash
    npm install
    ```

2.  Start the server (using `yarn`)

    ```bash
    	yarn serve
    ```

3.  Use `GET` or `POST` method

    > GET: localhost:5000/count/MCWCM

    > POST: localhost:5000/count

    > { "payload": "MCWCW"}

4.  Check tests

    ```bash
    yarn test  museumService
    ```
